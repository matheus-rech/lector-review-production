# Lector Hooks Available

Based on the GitHub repository, the following hooks are available in @anaralabs/lector:

## Hooks Found

1. **useAnnotationTooltip.ts** - For annotation tooltips
2. **useAnnotations.ts** - For PDF annotations
3. **useDpr.tsx** - For device pixel ratio
4. **usePDFLinkService.tsx** - For PDF link handling
5. **usePdfOutline.tsx** - For PDF outline/table of contents
6. **usePdfPageNumber.tsx** - For page number management
7. **useSelectionDimensions.tsx** - For text selection dimensions
8. **useThumbnail.tsx** - For page thumbnails

## Hook Folders

- **document/** - Document-related hooks
- **layers/** - Layer-related hooks
- **pages/** - Page-related hooks
- **search/** - Search-related hooks
- **viewport/** - Viewport-related hooks
